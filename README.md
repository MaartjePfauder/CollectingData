# CollectingData
Repository for the course 'collecting data' of the master's program 'Digital Humanities'
 
## Wine Dataset Practice
This repository contains the UCI Wine dataset for practising GitHub.

### Variables

| Column | Variable Name             | Description |
|---------|---------------------------|-------------|
| 1 | Alcohol | Alcohol content of the wine |
| 2 | Malic acid | Amount of malic acid |
| 3 | Ash | Ash content in the wine |
| 4 | Alcalinity of ash | Alkalinity of the ash |
| 5 | Magnesium | Magnesium content |
| 6 | Total phenols | Total phenolic compounds |
| 7 | Flavanoids | Amount of flavanoids |
| 8 | Nonflavanoid phenols | Amount of non-flavanoid phenols |
| 9 | Proanthocyanins | Amount of proanthocyanins |
| 10 | Color intensity | Intensity of the color |
| 11 | Hue | Hue of the wine |
| 12 | OD280/OD315 of diluted wines | Optical density ratio of diluted wines |
| 13 | Proline | Proline content |

### Files
- `wine.csv`: the dataset (renamed from `wine.data`)
- `wine.names`: dataset description (feature names and class labels)

### Source
Data source: [UCI Machine Learning Repository - Wine Data Set] (https://archive.ics.uci.edu/ml/datasets/wine)
